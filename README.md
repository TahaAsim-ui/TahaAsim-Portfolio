# Taha Asim Portfolio

Interactive static portfolio with an Interstellar-inspired mission-console theme.

## Run

Open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 5173
```

Then visit `http://127.0.0.1:5173`.
